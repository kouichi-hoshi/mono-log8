# テキスト・コンテンツ定義

アプリ内で表示されるテキスト（ウェルカムメッセージ、利用規約、案内文、アラート、通知など）を一元管理するドキュメントです。

## ウェルカムメッセージ

ログイン前の画面に表示するメッセージ。

```
メモとノートをまとめて管理しましょう

### このアプリについて

アプリ開発の学習のために制作した、
シンプルなメモ/ノートアプリです。

どなたでも無料でお試しいただけます。
機密情報等は入力しないようお願いします。

### 免責事項

このアプリは予告なく変更・削除される場合があります。
アプリのご利用に際しては、すべて利用者の責任においてご利用ください。


### Link

- 見出し要素にsr-onlyクラスを付ける
- SNSアイコンはlucide-reactを使用して設置する アイコンのみ、テキストラベルは不要 アイコンにはaria-labelを付ける
- お問い合わせはボタンとして設置する
- github (https://github.com/kouichi-hoshi)
- X (https://x.com/stella_d_tweet)
- お問い合わせ (https://stella-d.net/#contact)

```

## 通知・アラート

### 成功メッセージ

- 「保存しました」
- 「更新しました」
- 「投稿を削除しました」
- 「投稿を復元しました」
- 「投稿を完全に削除しました」
- 「すべての投稿を取得しました」

### エラーメッセージ

- 「内容を入力してください」
- 「内容は最大280文字までです」
- 「内容は最大20000文字までです」
- 「タグは最大10件まで追加できます」
- 「タグ名は最大32文字までです」
- 「ログインが必要です」
- 「認証エラーが発生しました」
- 「権限がありません」
- 「対象が見つかりません」
- 「入力内容に不備があります」
- 「サーバーエラーが発生しました」
- 「エラーが発生しました」

### 通知内アクション

- 「再試行」

## 削除確認モーダル

- 「n件の投稿を完全に削除しますか?」
- 「ごみ箱内のすべての投稿を完全に削除しますか?」
- 「この操作は取り消せません」

## ごみ箱の一括操作 UI ラベル

- 「ごみ箱を見る」
- 「表示されている投稿を選択」
- 「n件選択中」
- 「選択した投稿を削除」
- 「ごみ箱を空にする」

## 投稿操作ボタン ラベル

- 「削除」
- 「完全に削除」
