# 表示テキスト定義

アプリ内で表示されるテキスト（ウェルカムメッセージ、利用規約、案内文、アラート、通知など）を一元管理するドキュメントです。

## ウェルカムメッセージ

ログイン前の画面に表示するメッセージ。

```
メモとTODOリストをまとめて管理しましょう

### このアプリについて

アプリ開発の学習のために制作した、
シンプルなメモ/ToDoアプリです。

どなたでも無料でお試しいただけます。
機密情報等は入力しないようお願いします。

### 免責事項

このアプリは予告なく変更・削除される場合があります。
アプリのご利用に際しては、すべて利用者の責任においてご利用ください。


### Link
- github
- X
- お問い合わせ

```

## 通知・アラート

### 成功メッセージ

- 「保存しました」
- 「更新しました」
- 「投稿を削除しました」
- 「投稿を復元しました」
- 「すべての投稿を取得しました」

### エラーメッセージ

- 「内容を入力してください」
- 「認証エラーが発生しました」
- 「権限がありません」
- 「対象が見つかりません」
- 「入力内容に不備があります」
- 「サーバーエラーが発生しました」
- 「エラーが発生しました」

## 削除確認モーダル

- 「n件の投稿を完全に削除しますか？」
- 「ゴミ箱内のすべての投稿を完全に削除しますか？」
- 「この操作は取り消せません」

## ゴミ箱の一括操作 UI ラベル

- 「表示されている投稿を選択」
- 「n件選択中」
- 「選択した投稿を削除」
- 「ごみ箱を空にする」
