# Miro フレーム参照表

## 運用ルール

- 見た目（レイアウト/ワイヤー）は Miro を正とする（docs は挙動/状態/I/F を正とする）。
- 本表は「実装時にどのフレームを見るべきか」を最短で辿れることが目的。
- Miroのtitleには HTML entity（例: `&#xff08;`）が含まれる場合があるため、必要に応じて「表示名（人間向け）」と「取得title（原文）」を併記する。
- `.env.miro` の秘匿値（トークン等）は本文・ログに出力しない。
- Miro出力のリンク等は、必要がない限り本表に転記しない（frameIdとtitleは記載可）。
