# アプリ名

- 名称: Mono Log（モノログ）※キャッチフレーズにモノローグ
- keyなど: mono-log

---

# コンセプト

- **シンプルなメモ/ノートアプリ**
	- メモとは: 一行で書き切る短い走り書き/付箋/ToDo（シンプルなテキスト）
	- ノートとは: ページにしっかりと文章を書くイメージ（リッチテキスト + マークダウン風ショートカット入力をサポート）
	- 軽くて早い

- **タイムライン方式でメモとノートを素早く切り替え**
	- タイムライン方式: XのようなSNSによくある、投稿を立て一列で並べるシンプルなレイアウトのこと。無限スクロールで投稿を追加で読み込める

- **ストックよりもフローを重視した設計にする**
	- メモアプリはPCなら左サイドバーで投稿管理、右側にエディタ、スマフォなら投稿管理→エディタ、という構成がよくあるが、メモが増えると過去のメモの再利用が難しくなる。どこに何を書いたかわからなくなる。
	- 本アプリではそもそも再利用性（ストック）よりも、どんどん使い捨てる（フロー）、というコンセプトを重視する。

- **従来のストックの概念を再定義する**
	- よくあるディレクトリ階層でのファイル管理ではなく、AIによるレコメンドやサジェスト、検索により過去の投稿を利活用する（※これは将来導入予定の機能、初期フェーズでは実装しない）


- 機能をむやみに増やさず、複雑な操作を必要とせず、シンプルで軽量、直感的に使えることを重視する

---

# ターゲットユーザー

- 日々のメモやノートをシンプルに記録・管理したい人
- 複雑な機能は不要で、軽快に動作するアプリを求める人

---

# 基本要件

- SPAとして実装する
- PC用webブラウザ、及びスマフォ用webブラウザで利用できる
-  スマートフォンのホーム画面に追加することができる
- デザイン・レイアウト
	- レスポンシブデザインに対応する

---

# 機能要件

- **認証機能**
	- Googleアカウントによるログイン・ログアウトができる

- **投稿機能**
	- 投稿の作成・編集・ごみ箱投入ができること
	- 投稿にモード（メモ/ノート）を設定できる
		- モードにより入力フォームを切り替える
			- メモ：inputフィールド
			- ノート:リッチテキストエディタ
		- 投稿のモードは作成時に確定し、**作成後は変更できない**
	- 投稿一覧をタイムライン形式で表示できる
	- 無限スクロールで追加の投稿を読み込める
	- **お気に入り（スター）**
		- 投稿に「お気に入り」フラグを付けられる
	- **タグ付与**
		- 投稿にタグを付与できること
		- 投稿に付与したタグは削除ができる
		- ユーザーごとに異なるタグクラウドを保有する

	- **投稿一覧機能**
		- 投稿を一覧で表示できる
		- モードを切り替えても投稿の再読み込みはしない
			- タグ絞り込み条件の変更（追加/解除）とお気に入りON/OFFは再読込する
		- 一覧で表示されている投稿は、その場で編集・ごみ箱投入ができる

	- **投稿検索**
		- 投稿一覧をタグで投稿を絞り込める
			- 複数のタグを同時にアクティブにできる
		- 投稿一覧を「お気に入り」で絞り込める

- **モードによる表示切り替え機能**
	- 「メモ」「ノート」で投稿一覧を切り換えて表示する

- **ごみ箱機能**
	- 投稿を削除する機能は、
		- 1. ごみ箱に入れる
		- 2. ごみ箱で完全削除する
	- という二段階のステップを経て、ユーザーに削除の猶予を与える

---

# パフォーマンス

- 投稿一覧の初期表示が高速である
- 無限スクロールによる追加読み込みがスムーズである
- タイムラインのスクロール操作がストレスなく行える
- Core Web Vitals（本番, p75）が目標値を満たす
  - LCP <= 2.5s
  - CLS <= 0.1
  - INP <= 200ms

# 将来導入したい機能

- **コンシュルジュ機能（AIチャットボット）**
	- 検索サポート・レコメンド・サジェスト
	- キャラ付けして、マスコット的にチャットができる
	- 昔の偉人の「今日の格言」を解説するボット

- **外観変更**
	- システム設定／ライト／ダークの外観切り替え
	- 文字サイズ変更

- **検索機能（テキスト検索）**
	- 検索フィールドに文字列を入力し、検索ボタン押下で検索を実行する
	- タイトル、投稿本文、モード、タグのいずれかに検索文字列を含む投稿を一覧で表示する

- **投稿エクスポート機能**
	- 投稿データをエクスポートできること

- **下書きの自動保存/復元**
	- 入力中の投稿（未保存）を一時保存し、タブの再読み込みやブラウザ再起動後も復元できる
	- 再ログイン（401）時の入力保護にも利用する（例: `sessionStorage` / `localStorage`）

- **アカウント削除**
	ユーザーが自身のアカウントを削除できる

- **投稿自動削除**
	- ごみ箱投入日から一定日数が経過した投稿を自動削除する
