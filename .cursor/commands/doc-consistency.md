# /doc-consistency — ドキュメント整合性チェック

このスラッシュコマンドは、`.codex/skills/doc-consistency/SKILL.md` に定義された手順に従って、docs配下の整合性を点検します。

## 手順

以下のskillファイルを参照し、その手順に従って実行してください：

1. `.codex/skills/doc-consistency/SKILL.md` を読み込む
2. skillに定義された手順（Workflow）に従って実行する
3. 整合性チェックレポートを作成する
4. 修正が必要な場合は、ユーザーの許可を得てから実行する

## 注意事項

- `AGENTS.md` の禁止事項に抵触する提案はしない
- 修正前に「どの docs を真実とするか」が揺れる場合は、先に質問して確定する
- 必ずユーザーの許可を得てから修正を実行する
